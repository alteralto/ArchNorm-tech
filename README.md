# ArchNorm — техническая витрина

**AI-ассистент нормоконтроля строительной документации.** Находит применимую норму
(СП / СНиП / ГОСТ) за минуты и отвечает с **проверяемой ссылкой на конкретный пункт** —
а не «галлюцинацией». Работает **on-prem** (закрытый контур бюро) и в облаке из единой кодовой базы.

- 🌐 **Живое демо:** https://archnorm.ru
- 🔎 **Эта витрина (что под капотом):** https://alteralto.github.io/ArchNorm-tech/

> Полный исходный код — приватный (stealth-фаза), доступ по запросу под NDA.
> Корпус нормативов и внутренние данные не публикуются.

## Ценность
Проектировщик тратит 1–2 ч/день на ручной поиск по нормам, а поздно найденная ошибка стоит
бюро сотни тысяч рублей (платная переэкспертиза + переделка). ArchNorm даёт нужную норму за
7–10 минут с цитатой пункта — пригодно для нормоконтроля, а не «болталка».
Рынок подтверждён: прямой облачный аналог Norma.AI привлекает инвестиции, во Франции работает
ArchiNORM — а ниша **on-prem / закрытого контура (152-ФЗ)** свободна.

## Что внутри (для технической оценки)
- **RAG-пайплайн** без фреймворков: BGE-M3 + ChromaDB → reranker (BM25 / cross-encoder) →
  разрешение перекрёстных ссылок норм → LLM → ответ с источником (пункт / страница / цитата).
- **Метрики retrieval** (n = 132 вопроса): **Hit@3 94,7%**, Hit@1 78,8%, MRR 0,87, NDCG@10 0,88;
  + честная **отрицательная абляция** реранкера (включаем компонент только если eval подтверждает пользу).
- **Агенты** (оркестратор + доменные, напр. пожбезопасность) · **VLM** (Qwen3-VL) для схем и таблиц СП.
- **Два контура из одного кода:** локальный `llama.cpp` (offline, PII-редакция Presidio + РФ) и
  облако **Yandex Cloud** (YandexGPT).
- **Инфраструктура / MLOps:** Docker, GitHub Actions (pytest → Yandex Container Registry → SSH-деплой),
  ~25 тестов, Caddy TLS, безопасность API (X-API-Key, CORS, rate-limit), аналитика без PII.
